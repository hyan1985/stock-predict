# Stock Predict · 尾盘选股 Skill

基于尾盘 30 分钟分时形态 + 三步筛选的 A 股分析工具。

**在线分析：** https://stock-predict-we9pcfhnkrywlst7pziusn.streamlit.app/  
**源码仓库：** https://github.com/hyan1985/stock-predict

当用户提到「尾盘选股」、「stock-predict」、「单股涨跌分析」、「tail pick」等关键词时启用。

## 策略来源

抖音博主 **阳哥爱打板** 的尾盘30分钟选股法：
- A股 T+1 制度下，尾盘最后30分钟主力真实意图基本全暴露
- 尾盘买入、次日掌握主动权、降低隔夜风险

## 使用方法

当用户提供股票代码（如 600519、000858、002008）时：

1. **在线分析（推荐）**：打开 https://stock-predict-we9pcfhnkrywlst7pziusn.streamlit.app/ 输入代码或名称。

2. **本地 / 仓库内运行**：
   ```bash
   python tail_pick.py <代码1> <代码2> ...
   ```

2. **批量模式**（从文件读取）：
   ```powershell
   python tail_pick.py --list watchlist.txt
   ```

3. **解读输出**：
   - 🟢 强烈推荐 — 形态+筛选双通过，尾盘轻仓买入
   - 🟡 可以考虑 — 形态不错但条件未全过，小仓位试探
   - 🔴 建议回避 — 形态走坏，有出货嫌疑
   - ⚪ 暂不建议 — 条件不满足，空仓等待

4. **历史回测**（验证策略有效性）：
   ```powershell
   python tail_pick_backtest.py <代码1> <代码2> ... --days 20
   ```

5. **JSON 结果**也会保存到 `tail_pick_result.json`（实时）或 `tail_pick_backtest_result.json`（回测）。

## 策略核心逻辑

### 六种分时形态（+1 回测验证形态）

| 形态 | 描述 | 信号 |
|------|------|------|
| 1 | 横盘不动（洗盘吸筹） | 中性，等方向 |
| 2 | 先涨后跌跌破开盘（拉高出货） | 强烈回避 |
| 3 | 全天均线下方（反弹无力） | 回避 |
| 4 | 冲高回踩不破均线（承接有力） | 🟢 买入 |
| 5 | 小阳缓升+温和放量（主力吃货） | 🟢 买入 |
| 6 | 尾盘急拉远离均线（诱多出货） | 强烈回避 |
| 🄂 | 均线上方 — 尾盘稳稳站在均线上方（偏强） | 🟢 买入 |

> **第7个形态「均线上方(偏强)」**经回测验证（10只股票×40交易日）：收盘胜率 ~80%、平均次日收益 +2~5%。当尾盘大部分时间价格在分时均线（VWAP）上方运行，即使没有明显冲高回踩，也说明资金承接有力，次日大概率继续走强。

### 三步筛选法

**第一步**：涨幅 3%~5% — 强势但不极端
**第二步**（三不要）：
- 量比 < 1 → 不要
- 换手率 < 5% 或 > 10% → 不要
- 流通市值 < 50亿 或 > 200亿 → 不要
**第三步**（K线+分时确认）：
- 成交量温和放大（>105%且<250%）
- 5日线金叉（MA5 > MA10）
- 30日线趋势向上
- 尾盘创当日新高

### 最终评分

- 形态评分（权重60%）+ 筛选评分（权重40%）
- 80+ = 强烈推荐
- 60-79 = 可关注
- 40-59 = 可小仓位试探（均线上方偏强等形态）
- <40 = 暂不建议

## 回测验证

基于 mootdx 历史1分钟K线数据，对10只股票近40个交易日回测：

| 信号类型 | 收盘胜率 | 平均次日收益 | 高开概率 |
|----------|:--------:|:------------:|:--------:|
| 🟢 买入（形态4/5/均线上方偏强） | **~70%** | **+2.5%** | 38-60% |
| 🔴 回避（形态2/3/6） | 下跌仅33-36% | +1.9~2.1% | 低开43-64% |

**关键发现**：
- **买入端有效**：形态4（冲高回踩）胜率最高 69-70%，是策略基石
- **回避端在牛市中失灵**：下跌概率仅 33-36%，趋势市中"回避"信号不可盲信
- **尾盘急拉（形态6）是最可靠的空头信号**：样本少但准确率高
- **均线上方(偏强) 确认为买入信号**：多次回测收盘胜率 80-88%，已从"中性"升级为"买入"
- 回测脚本：`python tail_pick_backtest.py <代码> --days 20`

## 注意事项

- **仅在 14:30 后运行**效果最佳（尾盘数据完整）
- 盘后也可运行复盘，但当日判断的实战价值下降
- 每只股票之间自动间隔1.5秒，防止封IP
- **如果一只都没筛出来 → 空仓才是交易的王道**
- 策略仅供参考，不构成投资建议。理财有风险，投资需谨慎。
